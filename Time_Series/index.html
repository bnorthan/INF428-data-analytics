Moving Averages

  • Each point is the average of the previous n points
  • Used to get rid of noise

Windowed Average

  • Each point is the WEIGHTED average of the previous n points
  • Weighting is done by a "window" function
  • Weighting is done so more recent points are more important
  • Used to get rid of noise

Autocorrelation

  • Correlation within a time series
  • Can indicate periodic trends
  • For example temperatures over a period of years
  • Autocorrelation should be cyclical
