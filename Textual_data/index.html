Textual Data

  • Plain text is the most common form of data
  • NLTK -> Natural Language Toolkit
  • NLTK comes with sample texts called Corpora

Common Concepts

  • Corpus -> collection of written texts
  • Stopwords -> low information words
  • Guttenburg collection
  • Install NLTK --> conda install nltk
  • Download nltk corpus

import nltk
nltk.download()

Bag of words model

  • Create collection of words in document
  • Count the number of times each word appears
  • Use word counts for analsyis (ie detect spam in e-mail)

Preprocessing

  • Filter out very short words and single letter (a, I)
  • Filter out stopwords
    • common words with low information
    • ie the, all, just
  • Filter out names
  • Filter out numbers
