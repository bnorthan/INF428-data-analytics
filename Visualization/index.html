Plotting in Python with matplotlib

Basic Plotting example



import matplotlib.pyplot as plt

%matplotlib inline

x=[1,2,3,4,5]
y=[10, 20, 10, 30, 40]

plt.plot(x,y)
plt.show()

Plots can have the following

  • Legend describing each data series
  • Annotations for important points
  • Labels on horizontal and vertical axis
  • Title
  • Grid
