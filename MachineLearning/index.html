What is Machine Learning

  • Automated pattern analysis

Key Terminology

  • Features
  • Feature Vector
  • Classification
  • Regression
  • Clustering

Feature

  • Features are attributes
  • Numerical
    • Weight, age, wins, runs scored
  • Binary
    • Yes/No (1 or 0)
    • Male/Female
  • Enumeration/Categorical
    • Color, City, Education Level

Features Vector

  • Array of Features
  • Each feature converted to numerical value
    • Age Gender Education
    32 M High School
    27 F PhD

Features Vector

  • Array of Features
  • Each feature converted to numerical value
    • Age Gender Education
    32 0 1
    27 1 4

Classification

  • Identify a target variable
    • Type of bird
    • Credit risk category (high, low)
    • disease state
    • Filter out stopwords
  • Target variable is called a class
  • Steps
    • Train algorithm (give it known examples)
    • Apply to unknown (new) examples

Regression

  • Prediction of a numerical value
  • Linear regression
    • y: a output
    • x: one or more inputs
    • y=ax, or y=a1*x1+a2*x2+a3*x3

Clustering

  • No Target Value
  • Divide Data Into Groups
  • Unsupervised Learning
