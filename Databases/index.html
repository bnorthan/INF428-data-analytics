Relational Databases

  • Data organized in tables
  • Each row a record identified by a key
  • Each column holds a type of data
  • Rows in one table can be linked to other tables (relations)

SQL

  • Structured Query Language
  • SQL used to create and interact with databases
  • SQL is written in statements
    • 'SELECT name FROM players WHERE team='Blue Jays'
  • SQL can be accessed using other languages
  • SQL Tutorial

Example SQL Implementations

  • MySQL
  • SQLite
  • PostgreSQL

Using SQL in a website

  • A database program (MySQL, SQLLite, etc.)
  • Server side language (PHP, Python (Django, Flask)
  • Server side language talks to database with SQL
  • HTML/CSS used to show web pages in client (web browser)

Common SQL Commands

  • SELECT - extracts data from a database
  • UPDATE - updates data in a database
  • DELETE - deletes data from a database
  • INSERT INTO - inserts new data into a database
  • CREATE DATABASE - creates a new database

Common SQL Commands (continued)

  • ALTER DATABASE - modifies a database
  • CREATE TABLE - creates a new table
  • ALTER TABLE - modifies a table
  • DROP TABLE - deletes a table
  • CREATE INDEX - creates an index (search key)
  • DROP INDEX - deletes an index
